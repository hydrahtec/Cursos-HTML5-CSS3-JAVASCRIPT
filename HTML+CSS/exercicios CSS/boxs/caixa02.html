/-- nav : Essa teg representa um elemento de navegação no site, ele não suporta texto diretamente escrito nele (verificar se parágrafos também entrao nessa regra). É usado para colocar botões ou links. tipo box-level da classe fluxo e seus filhos tambem são de fluxo. para editar style: "nav {}" para editar os filhos seguir seletor: "nav a {}". Atributos: todas as tags do HTML podem receber atributos. eles são declarados no começo ou na abertura das tags. Eles podem liberar recursos ou funções diferentes da tag. --Escopo global (atributos que podem ser usadas em várias tags): * acsiscy: da um foco especial em uma parte específica do site; * class: * contented-editable: * data--: armazenar um valor ou informação dentro da tag. é possível usar esse atributo várias vezes no código. * * lang: específica qual a linguagem que será usada, seja no site como um rodo ou em partes específicas. * --/ Trabalhando com caixas em CSS
