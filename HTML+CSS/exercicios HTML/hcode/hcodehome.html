Atributos globais

Atributos globais, são atributos que todas as tags podem ter em comum.
Eles fornecem informações adicionais sobre um eletento e devem ser definidos na tag de abertura do elemento.
Os atributos podem ser expresos em pares nome/valor (ex: nome="valor" ) ou anepas o nome;

  • Accesskey
  • Class
  • Contenteditable
  • Data
  • Dir
  • Draggable
  • Drop zone-js
  • Drop Zone
  • Hidden
  • Id
  • Lang
  • Spell check
  • Style
  • Tab Index
  • Title
  • Translate
