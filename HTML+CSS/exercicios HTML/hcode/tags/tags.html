Tags da linguagem HTML 5

Confira abaixo todas as tags usadas na linguagem HTML 5

Tag Title

Éssa tag nomeia as paginas do seu site; Cada pagina deve ter um title unico que explique do que se trata a pagina; o title fica visivel na janela do navegador e talves num preview; no icone dos atalhos do navegador; herda todos os atributos globais e seu style: dispaly none.

Tag 'meta'

Essa tag define um metadato da página; Esses metadados serão usados por navegadores, mecanismos de pesquisa e outros serviços da web; eles devem ser declarados dentro do 'head' da página; style: none

Tag 'p'

Essa tag é usada para colocar textos na pagina, comentaros, posts...

Tag 'h'

A tag H é usada para definir os titulos do documento. Ela varia de nivel 1 á 6 (h1, h2, h3, h4, h5 e h6), sendo o 1 mais importante e o 6 menos importante. É recomendado que as páginas tenham somente um h1 por página pois ele representa o titulo principal da página e será muito importante para os mecanismos de busca. Os hs de 2 á 6 podem ser usados livremente respeitando sua ordem de importancia.

Tag 'style'

Define os esctilos CSS da página ou de um elemento espedifico. Posusi os atributos 'media' (que define um estilo de CSS especifico para diferentes aplicações como por exemplo uma impressão - media="print") e 'type' (que define qual o tipo de midia, no caso o padrão é tex/css)

Tag 'script'

Essa tag serve para inserir scipts no html. É mais usada para colocar codigos JavaScript. É recomendado colocar os codigos JavaScript em um arquivo externo e chama-lo dentro do html. Os scripts podem travar a rederização do documento, então recomenda-se que eles sejam executados por ultimo, para isso deve-se colocar os scripts no final do documento antes do termino da tag body (/body) ou colocar o atr defer (usado para rodar os scripts por ultimo)

Tag 'div'

Essa tag é usada para dividir um conteudo na página ou agrupar tags do documento, não tem valor semantico e serve basicamente para colocar estilos css em uma parte espefica do documento.

Tag 'span'

Essa tag serve basicamente para agrupar texto e colocar um estilo css. Ela não tem estilo e não possui valor semantico.

Tag link

Serve para lincar um arquivo externo. pode ser espeficamente um arquivo externo mas tabem pode ser um site ou outas paginas. É comumente usado para lincar arquivos css, favicon e scripts. recomenda-se usar na tag 'head'

Tag 'noscript'

A tag define um conteudo alternativo para uruarios que desativarem scripts no navegador ou quando os scripts não forem suportados. Nesse caso ira mostrar somente o html eu estiver dentro dessa tag, tambem é possivel definir estilos de css especificamente para esses casos.

Tag 'embad'

Usada para definir um conteudo externo embutido que será reproduzido dentro de um quadro dentro do doc.

Tag 'object'

Tag usada para definir um objeto incorporado no doc. É possivel incorporar audio, video, applets java activex, pdf e etc. Tabem é possivel passar parametros usando a tag 'param' que será vista mais a frente.
