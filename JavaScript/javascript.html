Curso JavaScript

Essa linguagem de programação é a mais usada no mundo quando o assunto é desenvolvimento de applicações para Web. É recomendado que todos os codigo JavaScript estejam localizados dentro da tag script e que essa tag esteja no fim do documento antes do fechamento do corpo da página (/body)

Variaveis

Uma varialvel é um espaço reservado na memoria do computador para se armazer informações que serão usadas em algum momento dentro do codigo. As variaveis se devidem em tipos, tambem conhecidos como tipos primitivos.

Importante:

  1. Maiusculas e minusculas fazem diferença.
  2. Ao se declarar uma varialvel não é necessario especificar seu tipo, a propria linguagem consegue fazer a definição.
  3. Utilize nomes que tenham a ver com o conteudo que será armazenado.

Tipos de variaveis (Primitivas)

  • Number

    • Não distingue valoreres numericos inteiros ou reais (fracionados, positivos ou negativos, essa variavel ingloba tudo)

    • infinity
    • NaN - (Significa que o valor não é um número.)

  • String

    • Trata-se de uma cadeia de caracteres, não necessariamente precisa ser um texto. Exemplo: o CPF é caracterizado como String pois possui alem dos numeros, a presença de pontos e traço.

    • .length - (Retorna quantos caracteres a string tem.)
    • .toUpperCase() - (Converte todos os caracteres para maiusculas)
    • .toLowerCase() - (Converte todos os caracteres para minusculas)
  • Bulean

    • Essa variavel resebe os valores 'verdadeiro' ou 'falso'

  • Null
  • Undefined
  • Object
    • arrey - ()
  • Function

Conversão de tipos de variaveis

Conversão String => Number

  • Number.ParseInt(valor) - (Converte o valor da variavel para Numero inteiro (o 'N' e o 'I' do name var são maiuculas))
  • Number.ParseFloat(valor) - (converte o valor para numero real)
  • Number - (o proprio JavaScript irá fazer a conversão seguindo o tipo de numero(Int ou Float); obs: se precisar forçar um tipo numerico especifico, use os comandos 1 e 2, se não houver a necessidade, use a opção 3, ela será suficiente)

Conversão Number => String

  • String(namevar) - (Converte o valor para o tipo string)
  • namevar.toString() - (Coloque o nome da varialvel seguido do comando.)

Dicas e macetes

Operador [+]

Ele pode agir como adição ou concatenação (usado para juntar textos com variaveis ou variaveis com variaveis qunado queremos mostralas varios dados na tela). Se o valor de todas as variaveis forem numericos (será realizada a operação de adição) mas se uma das variaveis for do tipo string (Será realizado a concatenação)

Metodo novo

Ao inves de usar o operado [+] para concatenar a exibição de textos e variaveis, o que deixa o codigo extenso e poluido, agora é possivel usar um recurso diferente que deixa o codigo mais limpo e organizado. Chama-se '', ele consiste em escrever o conteudo (textos e variaveis) que queremos apresentar ao usuario, todo dentro de uma mesma estrutura: window.alert(`Texto ${var} texto, ${var} texto texto!!`). Neste caso todo o conteudo é posto dentro crases e as variaveis são chamadas conforme a necessidade sem precisar fazer a separação uma da outra, basta colocar as variaveis dentro da estrutura ${}
