Variaveis

Uma varialvel é um espaço reservado na memoria do computador para se armazer informações que serão usadas em algum momento dentro do codigo. As variaveis se devidem em tipos, tambem conhecidos como tipos primitivos.

Importante:

  1. Maiusculas e minusculas fazem diferença.
  2. Ao se declarar uma varialvel não é necessario especificar seu tipo, a propria linguagem consegue fazer a definição.
  3. Utilize nomes que tenham a ver com o conteudo que será armazenado.

Tipos de variaveis (Primitivas)

  • Number

    • Não distingue valoreres numericos inteiros ou reais (fracionados, positivos ou negativos, essa variavel ingloba tudo)

    • infinity
    • NaN - (Significa que o valor não é um número.)

  • String

    • Trata-se de uma cadeia de caracteres, não necessariamente precisa ser um texto. Exemplo: o CPF é caracterizado como String pois possui alem dos numeros, a presença de pontos e traço.

    • .length - (Retorna quantos caracteres a string tem.)
    • .toUpperCase() - (Converte todos os caracteres para maiusculas)
    • .toLowerCase() - (Converte todos os caracteres para minusculas)
  • Bulean

    • Essa variavel resebe os valores 'verdadeiro' ou 'falso'

  • Null
  • Undefined
  • Object
    • arrey - ()
  • Function
